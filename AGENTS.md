# opbstack — รายการ Skill ทั้งหมด (Agent Index)

> **โปรเจกต์**: opbstack — Thai SME One Person Business AI Agent Stack  
> **ผู้สร้าง**: Apipoj Piasak | [data-espresso.com/apipoj](https://data-espresso.com/apipoj)  
> **รูปแบบ**: 10 Skills สำหรับผู้ประกอบการไทย One Person Business เรียกใช้งานผ่าน `/command`  

---

## วิธีใช้ (How to Use)

ใช้งานได้บน **Claude Code**, **Cursor**, และ **ChatGPT** (รวมถึง Gemini) โดยเรียก Skill ผ่านคำสั่ง `/` ตามด้วยชื่อ Skill:

```
/strategist  → วางแผนธุรกิจ
/brand       → สร้างแบรนด์
/build       → สร้างเว็บไซต์หรือระบบ
```

### วิธีเรียกใช้งาน (How to Invoke Skills)

| เครื่องมือ (Tool) | วิธีเรียก (Method) | ตัวอย่าง (Example) |
|---|---|---|
| **Claude Code** | พิมพ์ `/skillname` ตามด้วยคำสั่ง | `/strategist ช่วยวิเคราะห์ตลาดกาแฟ specialty ในกรุงเทพ` |
| **Cursor** | พิมพ์ `/skillname` ในช่อง AI Chat | `/brand สร้าง brand identity ร้านขนมไทย` |
| **ChatGPT** | ใช้ Custom GPT หรือพิมพ์ prompt ระบุ skill | `ใช้ skill /content เขียนแคปชั่นขายสินค้า organic` |
| **Gemini** | พิมพ์ prompt ระบุ skill ต้องการ | `ใช้ /data วิเคราะห์ยอดขายรายเดือน` |

## Flow การทำงาน (Workflow Flow)

แนะนำให้ใช้ Skill ตามลำดับนี้สำหรับผู้ประกอบการที่เริ่มต้นใหม่:

```
┌─────────────────────────────────────────────────────────────────────┐
│                                                                     │
│   [1] /strategist ──▶ [2] /brand ──▶ [3] /build                    │
│        วางแผน           สร้างแบรนด์      สร้างระบบ                   │
│                                                                     │
│        │                │               │                           │
│        ▼                ▼               ▼                           │
│   [4] /market ──▶ [5] /content ──▶ [6] /data                        │
│      การตลาด           คอนเทนต์        วิเคราะห์ข้อมูล                 │
│                                                                     │
│        │                │               │                           │
│        ▼                ▼               ▼                           │
│   [7] /serve ──▶ [8] /admin ──▶ [9] /money                         │
│      บริการลูกค้า       จัดการร้าน       การเงิน                      │
│                                                                     │
│        │                │               │                           │
│        ▼                ▼               ▼                           │
│              [10] /coach ──▶ กลับไปปรับปรุงใหม่                       │
│                โค้ชและพัฒนา                                          │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

### รายละเอียดแต่ละ Phase (Phase Details)

| Phase | Skills | เป้าหมาย (Goal) | ระยะเวลาโดยประมาณ |
|---|---|---|---|
| **Phase 1: วางรากฐาน** (Foundation) | `/strategist` → `/brand` → `/build` | มีธุรกิจที่ชัดเจนและระบบพร้อมใช้งาน | สัปดาห์ที่ 1-2 |
| **Phase 2: โตและขาย** (Growth & Sales) | `/market` → `/content` → `/data` | มีลูกค้าและสร้างรายได้ | สัปดาห์ที่ 3-6 |
| **Phase 3: ดูแลและปรับปรุง** (Operations) | `/serve` → `/admin` → `/money` | ระบบร้านค้าและการเงินมั่นคง | สัปดาห์ที่ 7-8 |
| **Phase 4: พัฒนาตนเอง** (Self-Development) | `/coach` | รักษากำลังใจและพัฒนาอย่างต่อเนื่อง | ตลอดเวลา |

## การใช้งานแบบไหลลื่น (Seamless Integration)

### ตัวอย่างการเชื่อม Skill ต่อกัน (Example: Skill Chaining)

```
/strategist → วิเคราะห์ตลาดร้านกาแฟ specialty ในกรุงเทพ
    ↓ (เอาผลลัพธ์ไปใช้ต่อ)
/brand → สร้างแบรนด์จากกลุ่มเป้าหมายที่วิเคราะห์มา
    ↓
/build → สร้างเว็บไซต์และระบบจองตามแบรนด์ที่กำหนด
    ↓
/market → วางแผนการตลาดเปิดตัวร้าน
    ↓
/content → เขียนคอนเทนต์โปรโมทเปิดตัว
```

### การใช้งานแบบวนรอบ (Continuous Loop)

```
/data (วิเคราะห์ยอดขาย) → /content (ปรับคอนเทนต์) → /market (ปรับแผน)
    ↑                                                    │
    └────────────────── /coach (ประเมินและปรับ) ←────────┘
```
