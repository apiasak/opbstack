## 1. บทนำ: thaismestack คืออะไร? (What is thaismestack?)

**thaismestack** คือ ชุดเครื่องมือ AI Agent 10 Skills ออกแบบมาสำหรับ **ผู้ประกอบการไทยแบบ One Person Business** — คนที่ทำธุรกิจคนเดียว ไม่มีทีม ไม่มีเงินจ้างใคร แต่ต้องการทำทุกอย่างให้ได้เหมือนมีทีมสนับสนุน

### ทำไมต้อง thaismestack?

ผู้ประกอบการไทยส่วนใหญ่เจอปัญหาเหล่านี้:

- **ไม่มีทีม** — ทำเองทุกอย่าง ตั้งแต่ขายของ ทำบัญชี ถึงการตลาด
- **ไม่มีงบ** — อยากได้ผลลัพธ์ดีแต่ไม่มีเงินจ้างเอเจนซี่
- **ไม่มีเวลา** — 24 ชั่วโมงไม่พอใช้จริงๆ
- **ไม่รู้ว่าจะเริ่มจากไหน** — มีไอเดียแต่ไม่มีแผน

**thaismestack** มาช่วยแก้ปัญหานี้โดยให้คุณมี "ทีม virtual" 10 คน ที่พร้อมช่วยงานได้ทันที ผ่าน AI Chat ที่คุณใช้อยู่แล้ว

### 10 Skills ที่คุณจะได้รับ

| Skill | ทำหน้าที่เหมือน... |
|---|---|
| `/strategist` | ที่ปรึกษาธุรกิจ (Business Consultant) |
| `/brand` | เอเจนซี่โฆษณา (Ad Agency) |
| `/build` | โปรแกรมเมอร์ (Developer) |
| `/market` | นักการตลาด (Marketing Manager) |
| `/content` | นักเขียน (Copywriter) |
| `/data` | นักวิเคราะห์ข้อมูล (Data Analyst) |
| `/serve` | ฝ่ายบริการลูกค้า (Customer Service) |
| `/admin` | เลขานุการ (Administrator) |
| `/money` | นักบัญชี (Accountant) |
| `/coach` | โค้ชส่วนตัว (Life Coach) |

## 3. การติดตั้ง (Installation)

### วิธีที่ 1: Claude Code (แนะนำ)

```bash
# 1. ติดตั้ง Claude Code (ต้องมี npm/Node.js ก่อน)
npm install -g @anthropics/claude-code

# 2. เข้าไปในโฟลเดอร์โปรเจกต์ของคุณ
cd /path/to/your/project

# 3. เปิด Claude Code
claude

# 4. ใช้ Skill ได้เลย!
/strategist ช่วยวิเคราะห์แผนธุรกิจร้านกาแฟ
```

> หมายเหตุ: Claude Code ปัจจุบันไม่ได้อยู่ในสถานะ “free beta” แบบข้อมูลเก่าแล้ว โดยทั่วไปจะใช้ผ่าน Pro/Max subscription หรือ API billing ตาม account ที่คุณล็อกอินไว้

**ขั้นตอนพร้อมคำอธิบายหน้าจอ (Step-by-Step with Screenshots Description):**

1. **เปิด Terminal** — เปิด Terminal (Mac/Linux) หรือ Command Prompt (Windows)
2. **รันคำสั่งติดตั้ง** — พิมพ์ `npm install -g @anthropics/claude-code` แล้วกด Enter
3. **ล็อกอิน** — ระบบจะเปิดหน้าเว็บให้ล็อกอินด้วยบัญชี Anthropic
4. **สร้างโฟลเดอร์โปรเจกต์** — `mkdir my-business && cd my-business`
5. **เปิด Claude Code** — พิมพ์ `claude` แล้วกด Enter
6. **เริ่มใช้ Skill** — พิมพ์ `/strategist` ตามด้วยคำสั่งที่ต้องการ

### วิธีที่ 2: Cursor

```bash
# 1. ดาวน์โหลด Cursor จาก cursor.com
# 2. ติดตั้งและเปิดโปรเจกต์
# 3. เปิดแท็บ AI Chat (Cmd+L หรือ Ctrl+L)
# 4. พิมพ์ /strategist แล้วตามด้วยคำสั่ง
```

### วิธีที่ 3: Paperclip (สำหรับ multi-agent workflow)

```bash
# 1. bootstrap local instance
npx paperclipai onboard --yes

# หรือ one-command bootstrap + run
pnpm paperclipai run

# 2. เปิด UI ที่ http://localhost:3100
# 3. สร้าง company และ CEO agent
# 4. คัดลอก skills ของ thaismestack ไป ~/.paperclip/skills/
# 5. อ่านคู่มือเต็มใน setup-paperclip.md
```

**Paperclip เหมาะเมื่อไหร่**
- อยากมีหลาย agent แยกหน้าที่จริง ๆ
- อยากควบคุม budget ของแต่ละ agent
- อยากมี issue / approval / audit trail
- อยากแยก strategist, content, build, data เป็นคนละ lane

> หมายเหตุ: docs ปัจจุบันของ Paperclip ใช้ flow แบบ UI-first หลัง `onboard/run` และ built-in adapter names รุ่นใหม่จะเป็น `claude_local`, `openclaw_gateway`, `hermes_local`, `process` เป็นต้น

### วิธีที่ 4: ChatGPT

```
# 1. เปิด ChatGPT Plus
# 2. ไปที่ "Explore GPTs"
# 3. สร้าง Custom GPT ใหม่
# 4. ใส่คำสั่ง (instructions) จากไฟล์ SKILL.md ของแต่ละ Skill
# 5. บันทึกและใช้งาน
```

### วิธีที่ 5: OpenClaw / Hermes Agent

ถ้าคุณอยากใช้ runtime ที่ยืดหยุ่นกว่าแบบ “agent จริง”:

- **OpenClaw** เหมาะกับงานที่ต้องเชื่อมหลายช่องทาง chat, plugin, gateway, และ multi-provider model routing
- **Hermes Agent** เหมาะกับงานที่ต้องใช้ persistent memory, self-learning, และ continuity ข้าม session

> สองตัวนี้เป็น open-source runtime แต่ยังมีค่า model/provider ถ้าคุณไม่ได้รัน local model เอง
>
> ถ้าจะใช้กับ BytePlus ModelArk หรือ integration ล่าสุด ให้ดูคู่มือย่อยใน repo โดยตรง เพราะ model IDs, pricing, และ endpoint เปลี่ยนเร็ว

### 4.1 `/strategist` — นักวางแผน (Strategist)

> **หน้าที่**: วิเคราะห์ธุรกิจ วางกลยุทธ์ หาโอกาสตลาด และวางแผนระยะยาว

**เมื่อไหร่ควรใช้ (When to Use)**
- เริ่มธุรกิจใหม่ ยังไม่รู้จะเริ่มจากไหน
- อยากรู้ว่าตลาดที่สนใจมีการแข่งขันยังไง
- ต้องการวิเคราะห์ SWOT ของธุรกิจตัวเอง
- อยากวางเป้าหมายรายเดือน/รายปี

**ตัวอย่าง Prompt (Example Prompts)**

```
/strategist วิเคราะห์ตลาดร้านกาแฟ specialty ในกรุงเทพ กลุ่มเป้าหมายอายุ 25-35
```

```
/strategist ช่วยทำ SWOT Analysis ธุรกิจขายเสื้อผ้ามือสองออนไลน์
```

```
/strategist วางแผน 3 เดือนแรกสำหรับธุรกิจรับทำเค้กวันเกิด เริ่มจากศูนย์
```

**เคล็ดลับ (Tips)**
- บอกข้อมูลธุรกิจตัวเองให้มากที่สุด — ยิ่ง AI รู้จักคุณมาก ยิ่งแม่น
- ถามต่อไปเรื่อยๆ ไม่ต้องกลัว — "แล้วถ้าแบบนี้ล่ะ?" "ช่วยขยายความข้อ 2"
- ขอให้ AI สรุปเป็น bullet points หรือตารางเพื่อง่ายต่อการอ่าน

### 4.2 `/brand` — สร้างแบรนด์ (Brand Builder)

> **หน้าที่**: สร้าง Brand Identity, ตั้งชื่อร้าน, ออกแบบโลโก้, กำหนด Tone of Voice

**เมื่อไหร่ควรใช้ (When to Use)**
- ยังไม่มีชื่อร้าน หรืออยากเปลี่ยนชื่อใหม่
- ต้องการสร้างเอกลักษณ์แบรนด์ให้จดจำได้
- อยากได้ Tone of Voice ที่ชัดเจน
- ต้องการสร้าง Brand Story ที่น่าสนใจ

**ตัวอย่าง Prompt (Example Prompts)**

```
/brand ช่วยคิดชื่อร้านขนมไทยโบราณ ที่ฟังดูมีเอกลักษณ์ จดจำง่าย
```

```
/brand สร้าง Brand Identity ร้านกาแฟ specialty สไตล์มินิมอล กลุ่มเป้าหมายวัยทำงาน
```

```
/brand เขียน Brand Story ร้านขายเสื้อผ้าที่ใช้ผ้าไทยตัดเย็บ ส่งต่อภูมิปัญญา
```

**เคล็ดลับ (Tips)**
- บอกความรู้สึก/อารมณ์ที่อยากให้ลูกค้าได้รับ — "อยากให้รู้สึกอบอุ่นเหมือนอยู่บ้าน"
- ระบุกลุ่มเป้าหมายชัดเจน — AI จะช่วยคิดได้ตรงใจกว่า
- ขอหลายตัวเลือก — "ช่วยเสนอ 5 ชื่อร้าน พร้อมเหตุผล"

### 4.3 `/build` — สร้างระบบ (Builder)

> **หน้าที่**: สร้างเว็บไซต์, Landing Page, ระบบจอง, ร้านค้าออนไลน์

**เมื่อไหร่ควรใช้ (When to Use)**
- อยากมีเว็บไซต์เป็นของตัวเอง
- ต้องการระบบจองนัดหมาย
- อยากสร้าง Landing Page ขายสินค้า
- ต้องการระบบร้านค้าออนไลน์ง่ายๆ

**ตัวอย่าง Prompt (Example Prompts)**

```
/build สร้าง Landing Page ร้านกาแฟ มีรูปร้าน รายการเครื่องดื่ม ที่อยู่ เปิด-ปิด
```

```
/build สร้างระบบจองคิวร้านตัดผม ที่ลูกค้าเลือกวันและเวลาได้
```

```
/build สร้างเว็บไซต์ร้านขายขนม มีหน้าแสดงสินค้า ตะกร้า และฟอร์มสั่งซื้อ
```

**เคล็ดลับ (Tips)**
- บอกรายละเอียดทุกอย่างที่ต้องการ — ยิ่งละเอียดยิ่งใกล้เคียง
- ขอให้ AI สร้างเป็น HTML/CSS/JS ธรรมดาก่อน — ง่ายต่อการแก้ไข
- ทดสอบบนมือถือเสมอ — ลูกค้าไทยส่วนใหญ่ใช้มือถือ

### 4.4 `/market` — การตลาด (Marketer)

> **หน้าที่**: วางแผนการตลาด, โฆษณา, Social Media, SEO, Influencer

**เมื่อไหร่ควรใช้ (When to Use)**
- เปิดร้านใหม่ อยากให้คนรู้จัก
- ยอดขายนิ่ง อยากหาวิธีดึงลูกค้าใหม่
- อยากลองโฆษณา Facebook/Instagram/TikTok
- ต้องการวางแผนการตลาดรายเดือน

**ตัวอย่าง Prompt (Example Prompts)**

```
/market วางแผนการตลาดเปิดตัวร้านกาแฟ งบ 10,000 บาท ระยะเวลา 1 เดือน
```

```
/market ช่วยคิดแคมเปญโปรโมชั่นสำหรับเทศกาลปีใหม่ ร้านขนม
```

```
/market วิธีทำ SEO ให้ร้านค้าออนไลน์ติดอันดับ Google
```

**เคล็ดลับ (Tips)**
- บอกงบประมาณชัดเจน — AI จะช่วยจัดสรรให้เหมาะสม
- ระบุช่องทางที่อยากโฟกัส — "เน้น Instagram และ TikTok"
- ขอ KPI ที่วัดผลได้ — "อยากได้ยอดขายเพิ่ม 20%"

### 4.5 `/content` — คอนเทนต์ (Content Creator)

> **หน้าที่**: เขียนคอนเทนต์, แคปชั่น, บทความ, สคริปต์วิดีโอ, ภาพกราฟิก

**เมื่อไหร่ควรใช้ (When to Use)**
- ต้องโพสต์ขายของแต่คิดแคปชั่นไม่ออก
- อยากเขียนบทความ SEO ให้เว็บไซต์
- ต้องการสคริปต์ถ่าย TikTok/Reels
- อยากได้ไอเดียคอนเทนต์รายสัปดาห์

**ตัวอย่าง Prompt (Example Prompts)**

```
/content เขียนแคปชั่นขายสินค้า organic สไตล์กันเอง ไม่เหมือนเซลล์
```

```
/content สร้างคอนเทนต์ 7 วัน สำหรับร้านกาแฟ แบบละเอียด พร้อมแฮชแท็ก
```

```
/content เขียนสคริปต์ TikTok 15 วินาที โปรโมทร้านขนม สไตล์ตลก
```

**เคล็ดลับ (Tips)**
- บอกโทนที่ต้องการ — "สไตล์เป็นกันเอง ใช้ภาษาพูด" หรือ "สไตล์ทางการ"
- ให้ตัวอย่างคอนเทนต์ที่ชอบ — "ชอบแบบร้าน @xxx"
- ขอหลายเวอร์ชั่น — "ช่วยเขียน 3 แบบให้เลือก"

### 4.6 `/data` — ข้อมูล (Data Analyst)

> **หน้าที่**: วิเคราะห์ข้อมูล, ยอดขาย, รายงาน, Dashboard, A/B Testing

**เมื่อไหร่ควรใช้ (When to Use)**
- มีข้อมูลยอดขายอยากรู้ว่าอะไรขายดีที่สุด
- อยากสร้างรายงานสรุปประจำเดือน
- ต้องการวิเคราะห์พฤติกรรมลูกค้า
- อยากรู้ว่าคอนเทนต์ไหนทำให้ยอดขายเพิ่ม

**ตัวอย่าง Prompt (Example Prompts)**

```
/data วิเคราะห์ยอดขายรายเดือน: มกราคม 50,000, กุมภาพันธ์ 45,000, มีนาคม 60,000
```

```
/data สร้างรายงานสรุปประจำสัปดาห์ จากข้อมูลออเดอร์ 100 ออเดอร์
```

```
/data ช่วยวิเคราะห์ว่าคอนเทนต์แบบไหนทำให้ยอดขายเพิ่มมากที่สุด
```

**เคล็ดลับ (Tips)**

### 4.7 `/serve` — บริการ (Customer Service)

> **หน้าที่**: ดูแลลูกค้า, ตอบคำถาม, จัดการร้องเรียน, สร้าง Loyalty

**เมื่อไหร่ควรใช้ (When to Use)**
- ลูกค้าถามคำถามซ้ำๆ อยากทำคำตอบสำเร็จ
- มีลูกค้าไม่พอใจ อยากรู้วิธีตอบกลับดีๆ
- อยากสร้างระบบ Loyalty ร้านค้า
- ต้องการวางมาตรฐานการบริการลูกค้า

**ตัวอย่าง Prompt (Example Prompts)**

```
/serve ช่วยเขียนคำตอบลูกค้าที่บ่นว่าส่งสินค้าช้า แบบใจเย็น แต่แก้ปัญหาได้
```

```

```


**เคล็ดลับ (Tips)**
- ให้ AI รู้บริบท — "ลูกค้าซื้อครั้งแรก" หรือ "ลูกค้าขาประจำ"
- เน้นความจริงใจ — ลูกค้าไทยรู้ว่าอะไรอำนวย อะไรจริงใจ

### 4.9 `/money` — การเงิน (Finance)

**เคล็ดลับ (Tips)**

- ขอสูตร Excel — เอาไปใช้คำนวณเองได้

### 4.10 `/coach` — โค้ช (Business Coach)

## 5. เทคนิคการ Prompt ให้ได้ผลดี (Prompting Tips)

### 5.1 บอก Context ให้ชัด (Be Specific)

❌ **ไม่ดี**: `/content เขียนแคปชั่นขายของ`

✅ **ดี**: `/content เขียนแคปชั่นขายเสื้อยืดลายไทย ราคา 390 บาท กลุ่มเป้าหมายวัยรุ่น สไตล์เท่ๆ กันเอง`

### 5.2 ให้ตัวอย่าง (Give Examples)

❌ **ไม่ดี**: `/content เขียนบทความ`

✅ **ดี**: `/content เขียนบทความเกี่ยวกับวิธีเลือกกาแฟคั่วกลาง สไตล์เหมือนบทความนี้: [แปะตัวอย่าง]`

### 5.3 ถามต่อไปเรื่อยๆ (Follow-up Questions)

```
/strategist วิเคราะห์ตลาดร้านกาแฟ specialty
    ↓
"ช่วยขยายเรื่องคู่แข่งหน่อย"
    ↓
"แล้ววิธีต่างตัวจากคู่แข่งยังไงดี"
    ↓
"ช่วยสรุปเป็นแผนปฏิบัติการ 1 เดือน"
```

### 5.4 ขอรูปแบบที่ต้องการ (Specify Output Format)

```
"ช่วยสรุปเป็น bullet points"
"ทำเป็นตารางให้หน่อย"
"เขียนแบบสั้นๆ ไม่เกิน 3 บรรทัด"
"ทำเป็น checklist ที่ทำได้เลย"
```

### 5.5 ใช้ภาษาที่เข้าใจง่าย (Use Simple Language)

ไม่ต้องใช้ศัพท์เทคนิค — พูดเหมือนคุยกับเพื่อน:

```
"ช่วยอธิบายแบบคนธรรมดาฟังรู้เรื่อง"
"อธิบายเหมือนสอนเพื่อนที่ไม่รู้เรื่องนี้เลย"
```

**การใช้ thaismestack**:

```
Week 1: /strategist
    ↳ "วิเคราะห์ตลาดร้านกาแฟ specialty ย่านอารีย์ กลุ่มเป้าหมาย 25-40 ปี"
    ↳ ได้: แผนธุรกิจ, กลุ่มเป้าหมาย, คู่แข่ง, จุดแข็งที่ควรโฟกัส

Week 1: /brand
    ↳ "สร้าง brand identity ร้านกาแฟ สไตล์มินิมอล เน้นความใส่ใจในทุกแก้ว"
    ↳ ได้: ชื่อร้าน "Slow Bar", โลโก้ concept, tone of voice, สีประจำแบรนด์

Week 2: /build
    ↳ "สร้าง Landing Page ร้าน Slow Bar มีเมนู ที่อยู่ ระบบจองโต๊ะ"
    ↳ ได้: เว็บไซต์พร้อมใช้งาน

Week 3: /market

    ↳ ได้: แผนการตลาดรายสัปดาห์, โปรโมชั่นเปิดตัว

Week 3: /content
    ↳ "สร้างคอนเทนต์ Instagram 7 วันแรก พร้อมแคปชั่นและแฮชแท็ก"

Week 4: /data

    ↳ ใช้วนไปตามความต้องการในแต่ละวัน
```

## 7. การแก้ปัญหาเบื้องต้น (Troubleshooting)

### ปัญหาที่พบบ่อย (Common Issues)

| ปัญหา | สาเหตุ | วิธีแก้ |
|---|---|---|
| **AI ไม่เข้าใจคำสั่ง** | พิมพ์คำสั่งไม่ชัดเจน | ระบุ Skill ชัดเจน `/strategist` ไม่ใช่ "ช่วยวิเคราะห์หน่อย" |
| **คำตอบยาวเกินไป** | ไม่ได้ระบุความยาว | เพิ่ม "สรุปสั้นๆ" หรือ "ไม่เกิน 5 ข้อ" |
| **AI ตอบไม่ตรงกับธุรกิจไทย** | ขาด context ตลาดไทย | ระบุ "ตลาดไทย" "ลูกค้าไทย" เสมอ |
| **ไม่รู้จะเริ่มจาก Skill ไหน** | ไม่มีแผน | เริ่มจาก `/strategist` เสมอ |
| **AI ตอบซ้ำๆ ไม่มีสาระ** | Prompt กว้างเกินไป | ใส่รายละเอียดเฉพาะเจาะจงมากขึ้น |
| **ไม่สามารถรันโค้ดได้** | ใช้ ChatGPT แทน Claude Code | Claude Code และ Cursor รันโค้ดได้ ChatGPT ให้คำแนะนำอย่างเดียว |
| **ข้อมูลล้าสมัย** | AI มี cutoff date | ระบุ "ข้อมูลล่าสุดที่คุณรู้" หรือใส่ข้อมูลปัจจุบันไปเอง |

### วิธี Reset การสนทนา (How to Reset)

ถ้า AI สับสนหรือตอบผิดแนว:

```
"เริ่มต้นใหม่ — ฉันจะใช้ Skill /strategist วิเคราะห์ธุรกิจร้านกาแฟ"
```

หรือเปิดหน้าสนทนาใหม่ (new chat) แล้วเริ่มต้นใหม่

## 8. คำถามที่พบบ่อย (FAQ)

**Q: ใช้ฟรีไหม? (Is it free?)**  
A: thaismestack เป็นชุดคำสั่ง (prompts) ที่ใช้ฟรี แต่คุณต้องมีบัญชี AI Chat (Claude Code, Cursor, หรือ ChatGPT Plus) ซึ่งบางอันฟรี บางอันเสียเงิน

**Q: ต้องเป็น programmer ไหม? (Do I need to code?)**  
A: ไม่ต้อง! ทุก Skill ใช้ผ่านการพิมพ์คำสั่งธรรมดา ส่วน `/build` ที่เกี่ยวกับเว็บไซต์ AI จะช่วยเขียนโค้ดให้ คุณแค่ copy ไปใช้

**Q: ใช้ Skill ไหนก่อน? (Which skill first?)**  
A: แนะนำให้เริ่มจาก `/strategist` เสมอ เพราะจะช่วยวางแผนว่าควรใช้ Skill อะไรต่อ

**Q: ใช้ได้กับธุรกิจอะไรบ้าง? (What businesses?)**  
A: ได้กับทุกธุรกิจ! ไม่ว่าจะร้านกาแฟ ขายของออนไลน์ รับฟรีแลนซ์ หรือบริการต่างๆ

**Q: AI ตอบผิดทำยังไง? (AI is wrong?)**  
A: บอก AI ว่า "ข้อนี้ไม่ถูกต้อง" และให้ข้อมูลที่ถูกต้องไป AI จะปรับปรุงคำตอบ
