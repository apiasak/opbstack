> ใช้เป็นแนวทางว่าควรตอบละเอียดแค่ไหน แล้วปรับให้เข้ากับธุรกิจของคุณ

## ส่วนที่ 1: ธุรกิจนี้คืออะไร? (What Business Is This?)

### 1.1 ชื่อธุรกิจ / ร้าน / แบรนด์
```
ชื่อ: Slow Bar (โซว บาร์)
สาขา/แฟรนไชส์:  ☐ ใช่  ☑ ไม่ใช่ (เป็นของตัวเอง)
```

### 1.2 ประเภทธุรกิจ
```
☑ อาหารและเครื่องดื่ม (F&B)
☐ แฟชั่นและเสื้อผ้า
☐ ความงามและสุขภาพ
☐ ของใช้และของตกแต่งบ้าน
☐ บริการ (Service)
☐ คอร์สเรียน / ให้คำปรึกษา
☐ ดิจิทัลและเทคโนโลยี
☐ อื่น ๆ: _________________________________
```

### 1.3 สินค้า/บริการหลักคืออะไร?
```
ขายอะไร: กาแฟ specialty (Single origin, Pour-over, Cold brew) 
         + ขนมโฮมเมด (Brownie, Basque cheesecake)
         + เมล็ดกาแฟคั่วสดขายเป็นถุง

ราคาโดยประมาณ: กาแฟ 90-180 บาท, ขนม 70-120 บาท, เมล็ด 250-350 บาท/ถุง

จุดเด่นของสินค้า/บริการ:
- ใช้เมล็ดกาแฟจากฟาร์มไทย (เชียงราย, เชียงใหม่) คั่วสดทุกสัปดาห์
- บาริสต้าเป็นคนทำเอง มีเรื่องเล่าให้ลูกค้าฟังทุกแก้ว
- บรรยากาศร้านเงียบสงบ เหมาะกับคนทำงาน/อ่านหนังสือ
```

### 1.4 ช่องทางการขายปัจจุบัน
```

☑ Instagram — @slowbarbkk
☑ LINE Official Account — @slowbar
☐ TikTok Shop

```

### 1.5 ระยะเวลาทำธุรกิจ
```
☐ ยังเป็นไอเดีย
☐ 0-6 เดือน
☑ 6 เดือน - 1 ปี (เปิดมา 8 เดือน)
☐ 1-3 ปี
☐ มากกว่า 3 ปี
```

## ส่วนที่ 2: ลูกค้าคือใคร? (Who Is the Customer?)

### 2.1 กลุ่มลูกค้าเป้าหมายหลัก
```
อายุ: 25 - 40 ปี
เพศ:  ☑ หญิง  ☑ ชาย  ☐ ทุกเพศ
รายได้ต่อเดือน: 30,000 - 80,000 บาท
อาชีพ: พนักงานออฟฟิศย่านสุขุมวิท, Freelancer, Creative professional
ที่อยู่:  ☑ กรุงเทพ  ☐ ต่างจังหวัด  ☐ ทั่วไทย  ☐ ต่างประเทศ
```

### 2.2 ลูกค้าซื้อเพราะอะไร?
```
ปัญหาที่ลูกค้ามีก่อนซื้อ: 
- อยากกินกาแฟดี ๆ แต่ไม่อยากไปไกล
- เบื่อกาแฟเชนใหญ่ อยากลองอะไรใหม่
- หาที่นั่งทำงานเงียบ ๆ มี Wi-Fi ดี

สิ่งที่ลูกค้าอยากได้หลังซื้อ:
- ได้ดื่มกาแฟที่มีเรื่องราว รู้ว่ามาจากไหน
- มีพื้นที่ส่วนตัวทำงาน/อ่านหนังสือ
- รู้สึกเหมือนเป็น "regular" ที่ร้านจำชื่อได้

อารมณ์/ความรู้สึกที่อยากให้ลูกค้ามี:
- "เหมือนมานั่งบ้านเพื่อนที่ชงกาแฟเก่ง"
- ผ่อนคลาย ไม่รีบ ไม่วุ่นวาย
- ภูมิใจที่ได้สนับสนุนกาแฟไทย
```

### 2.3 ลูกค้าซื้อที่ไหน / ยังไง?
```
ลูกค้าหาสินค้าคุณเจอได้ยังไง: 
- เดินผ่านหน้าร้าน
- Instagram (ดูรูปสวย ๆ)
- รีวิวใน Wongnai/Google Maps
- เพื่อนแนะนำ (Word of mouth)

เวลาซื้อมักจะเป็น: 
☑ ตอนเช้า (ก่อนเข้างาน 7-9 โมง)
☐ กลางวัน
☑ เย็น (เลิกงาน 4-6 โมง)
☐ ดึก
☑ วันหยุด (มานั่งทำงานยาว ๆ)
☐ ตลอดเวลา

ลูกค้าซื้อบ่อยแค่ไหน:  ☐ ซื้อครั้งเดียว  ☑ ซื้อซ้ำ  ☐ สมาชิก/Subscribe
```

### 2.4 คู่แข่งคือใคร?
```
คู่แข่งโดยตรง: 
- Roots (เชนใหญ่ มีหลายสาขา)
- Phil Coffee (specialty ใกล้เคียง)
- Ceresia Coffee Roasters (คู่แข่งระดับเดียวกัน)

คู่แข่งออนไลน์: 
- แบรนด์เมล็ดกาแฟออนไลน์ (Nine One Coffee, Mother Roaster)

สิ่งที่คู่แข่งทำดีกว่าคุณ:
- Roots มีสาขาเยอะ หาได้ทั่วไป
- Phil มีชื่อเสียงมาก่อน

สิ่งที่คุณทำดีกว่าคู่แข่ง:
- รู้จักลูกค้าส่วนตัว จำชื่อและรสนิยมได้
- บรรยากาศเงียบสงบกว่า (ไม่วุ่นวาย)
- มีเรื่องราวฟาร์มไทยเล่าให้ฟังทุกแก้ว
```

## ส่วนที่ 3: Brand Voice & Positioning

### 3.1 Personality ของแบรนด์
```
☐ มืออาชีพ จริงจัง
☐ เป็นกันเอง เหมือนเพื่อน
☐ ตลก ขำขัน
☐ หรูหรา ดูแพง
☑ เรียบง่าย มินิมอล
☐ กล้าดี เปรี้ยวจี๊ด
☑ อบอุ่น ดูแลเอาใจใส่
☐ อื่น ๆ
```

### 3.2 คำที่แบรนด์ใช้ / ไม่ใช้
```
คำที่ชอบใช้: 
- "คั่วสดทุกสัปดาห์"
- "จากฟาร์มถึงแก้ว"
- "ช้า ๆ แต่ดี"
- "กาแฟไทยที่คุณภาพระดับสากล"

คำที่ไม่ควรใช้:
- "เร็ว" (ไม่ใช่จุดขาย)
- "ถูก" / "ราคาประหยัด" (positioning ไม่ใช่ price leader)
- "เยอะ" (ไม่เน้นปริมาณ)

สำนวนที่เป็นเอกลักษณ์:
- "วันนี้คั่วอะไรดี?" (ทักทายลูกค้าประจำ)
- "เมล็ดใหม่มาแล้ว มาลองก่อนใคร" (สื่อสารกับลูกค้า)
```

### 3.3 จุดยืนของแบรนด์ (Brand Positioning)
```
ราคา:  ☐ ถูก  ☐ ปานกลาง  ☑ แพง  ☐ หรูหรา

คุณภาพ:  ☐ ใช้ได้  ☐ ดี  ☑ ดีมาก  ☐ ระดับพรีเมียม

ความเร็ว:  ☐ ส่งไว  ☐ ปกติ  ☑ ใช้เวลา (Made-to-order)

ความพิเศษ:  ☑ Handmade  ☐ นำเข้า  ☑ วัตถุดิบพิเศษ (กาแฟไทย single origin)
```

### 3.4 ค่านิยมของแบรนด์ (Brand Values)
```
3 คำที่อธิบายแบรนด์: 1. จริงใจ  2. ช้าแต่ชัวร์  3. ไทยแลนด์

สิ่งที่แบรนด์เชื่อมั่น:
"กาแฟไทยมีคุณภาพระดับโลก แค่ต้องมีคนเล่าเรื่องให้ฟัง"

สิ่งที่แบรนด์ไม่ยอมประนีประนอม:
- ไม่ใช้เมล็ดที่คั่วเกิน 7 วัน
- ไม่ปรุงแต่งรสชาติด้วยน้ำเชื่อม (ให้กาแฟพูดเอง)
```

## ส่วนที่ 4: ตัวตนของเจ้าของ / Agent Identity

### 4.1 ใครเป็นคนทำธุรกิจนี้?
```
ชื่อเล่น/ชื่อที่ใช้แสดงตน: ปอย (Ploy) — "Barista คนที่จะจำรสนิยมกาแฟคุณได้"

บทบาท:  ☑ ทำคนเดียวทุกอย่าง  ☐ มีทีมเล็ก ๆ  ☐ เป็นพาร์ทไทม์

ทำธุรกิจนี้เพราะ:
"อยากให้คนไทยภูมิใจในกาแฟไทย ไม่ต้องบูชากาแฟต่างชาติ"
```

### 4.2 สไตล์การทำงาน
```
☑ ชอบวางแผนละเอียดก่อนลงมือ
☐ ลงมือทำก่อน คิดทีหลัง
☑ ชอบทดลองไอเดียใหม่ ๆ (เมล็ดใหม่ วิธีชงใหม่)
☐ ชอบทำสิ่งเดิมให้ดีที่สุด
☐ ทำงานเร็ว เน้นปริมาณ
☑ ทำงานช้า เน้นคุณภาพ
☑ ทำตอนเช้า (เปิดร้าน 7 โมง ตื่นมาชงกาแฟตัวเองก่อน)
☐ ทำตอนกลางคืน
☐ ทำเป็นช่วง ๆ
☐ ทำเรื่อย ๆ ทุกวัน
```

### 4.3 จุดแข็งและจุดอ่อน
```
เก่งอะไรมากที่สุด: 
- รู้เรื่องกาแฟลึกซึ้ง (Q Grader)
- ชงกาแฟอร่อยทุกแก้ว
- พูดคุยกับลูกค้าเก่ง จำชื่อได้

สิ่งที่ยากที่สุดตอนนี้:
- ทำการตลาดออนไลน์ (ไม่ถนัดถ่ายรูป/ตัดวิดีโอ)
- บัญชีและเอกสาร (เกลียดตัวเลข)
- ต้องทำคนเดียวหมด ไม่มีเวลา

อยากให้ AI ช่วยอะไรมากที่สุด:
- เขียนแคปชั่น Instagram ที่ดูเป็นธรรมชาติ
- วางแผนการตลาดที่ไม่ต้องใช้เงินเยอะ
- ช่วยจัดการเอกสารและบัญชี
```

### 4.4 เป้าหมายส่วนตัวกับธุรกิจนี้
```
ระยะสั้น (3 เดือน):
- ยอดขายร้านเดือนละ 150,000 บาท
- ลูกค้า LINE OA 1,000 คน
- ขายเมล็ดกาแฟออนไลน์ได้เดือนละ 50 ถุง

ระยะกลาง (1 ปี):
- มีพนักงาน 1 คนช่วยในช่วงเย็น
- ขายเมล็ดกาแฟออนไลน์เป็นหลัก (ส่งทั่วไทย)
- รายได้รวม 300,000 บาท/เดือน

ระยะยาว (3 ปี):
- เปิดร้านที่ 2 ในย่านอื่น
- มีแบรนด์เมล็ดกาแฟเป็นของตัวเองที่รู้จักทั่วไทย
- รายได้ 1 ล้านบาท/เดือน

รายได้เป้าหมายต่อเดือน: 300,000 บาท (รวมหน้าร้าน + ออนไลน์)
```

## ส่วนที่ 5: ข้อมูลเพิ่มเติม (Optional)

### 5.1 อะไรที่ทำให้ธุรกิจนี้ "พิเศษ"?
```
ทุกแก้วที่ชง ปอยจะเล่าเรื่องฟาร์มที่ปลูกเมล็ดนั้นให้ลูกค้าฟัง
บางทีก็ชวนคุยเรื่องกาแฟ บางทีก็ปล่อยให้ลูกค้านั่งเงียบ ๆ
ร้านไม่มีเพลงดัง มีแต่เสียงหยดกาแฟและเสียงคนพิมพ์คีย์บอร์ด
```

### 5.2 คำขวัญ / Quote / Slogan
```
"ช้า ๆ ได้กาแฟดี" — Slow down, get good coffee.
```

### 5.3 มีอะไรอยากบอก AI อีกไหม?
```
ปอยเป็นคน introvert ที่พูดกับลูกค้าหน้าร้านเก่ง 
แต่พอเป็นออนไลน์แล้วไม่รู้จะพูดยังไง 
อยากได้เสียงที่ดูอบอุ่น ไม่ aggressive 
เหมือนเพื่อนชวนมากินกาแฟ ไม่ใช่ร้านค้าขายของ
```
